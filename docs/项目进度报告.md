### Agent 详情页功能

#### 功能概述

Agent 详情页提供了一个全面的界面来查看和管理单个 Agent 的详细信息，包括信用分评估、任务管理和高级筛选功能。

#### 主要特性

**1. Agent 基本信息展示**
- Agent 名称、ID、版本
- 状态徽章（空闲/活跃/忙碌/离线/握手失败/审计失败）
- 信任等级徽章（unknown/pending/trusted/restricted/revoked）
- 角色标签
- 功能描述和当前目标
- Endpoint、注册时间、最后活跃时间、成功率

**2. 信用分评估系统**

信用分基于 5 个维度计算，总分 0-100：

| 维度 | 权重 | 说明 |
|------|------|------|
| 任务完成率 | 30% | 已完成任务数 / 总任务数 |
| 响应延迟评分 | 25% | 基于平均延迟的分段评分 |
| 错误率评分 | 20% | (1 - 错误率) × 100 |
| 审计合规评分 | 15% | 基于信任等级 |
| 历史稳定性评分 | 10% | 基于成功率 |

每个维度显示：
- 得分（0-100）
- 权重百分比
- 详细说明
- 进度条可视化
- 趋势指示（up/down/stable）

**3. 任务统计卡片**
- 总任务数
- 已完成任务数
- 进行中任务数
- 待处理任务数
- 失败任务数
- 平均完成时间

**4. 任务管理 Tab**

四个 Tab 分别显示不同状态的任务：

- **正在进行**: `in_progress` 状态的任务
- **待处理**: `todo`, `blocked`, `waiting_confirmation` 状态的任务
- **失败任务**: `failed` 状态的任务
- **历史记录**: `done`, `cancelled` 状态的任务

每个 Tab 使用 DataGrid 展示，支持：
- 分页（10/20/50 条每页）
- 排序
- 列显示：任务ID、标题、状态、进度、发布方、开始/完成时间

**5. 任务操作功能**

**取消任务**:
- 适用状态: todo, in_progress, blocked, waiting_confirmation
- 操作: 将任务状态改为 cancelled
- 确认对话框: 提示用户谨慎操作
- 结果: 任务从当前 Tab 消失

**重试任务**:
- 适用状态: failed
- 操作: 将任务状态重置为 todo
- 确认对话框: 说明任务将重新执行
- 结果: 任务移动到"待处理"Tab

**6. 高级筛选和搜索**

可折叠的筛选面板，包含：

- **搜索框**: 按任务标题或 ID 模糊搜索
- **任务类型**: 下拉选择（数据处理/分析/生成/验证）
- **发布方**: 按 originator_id 过滤
- **日期范围**: 开始日期和结束日期选择器
- **清除筛选**: 一键重置所有条件

筛选条件变化时自动刷新数据。

#### 技术实现

**后端接口**:

1. `GET /api/web/agents/{agent_id}/detail`
   - 返回 Agent 完整信息
   - 包含信用分和统计数据

2. `GET /api/web/agents/{agent_id}/tasks/by-status`
   - 支持分页和高级筛选
   - 查询参数: status, page, page_size, sort_by, order, search, task_type, originator, date_from, date_to

3. `POST /api/web/agents/{agent_id}/tasks/{task_id}/cancel`
   - 取消指定任务

4. `POST /api/web/agents/{agent_id}/tasks/{task_id}/retry`
   - 重试失败任务

**前端组件**:

- `/src/admin-portal/src/pages/agents/AgentDetail.tsx` (566行)
- 路由: `/console/agents/:agentId`

#### 用户体验优化

1. **智能按钮显示**: 根据任务状态动态显示可用操作
2. **实时筛选**: 筛选条件变化时自动刷新
3. **防误操作**: 危险操作需要二次确认
4. **友好提示**: 清晰的成功/失败反馈
5. **响应式设计**: 适配不同屏幕尺寸
6. **加载状态**: 显示加载动画，避免用户困惑

#### 使用场景示例

**场景 1: 监控 Agent 健康状况**
```
1. 访问 Agent 详情页
2. 查看信用分总分和各维度得分
3. 识别低分维度，定位问题
4. 查看任务统计，了解工作负载
```

**场景 2: 处理卡住的任务**
```
1. 在"正在进行"Tab 找到长时间无进展的任务
2. 点击取消按钮
3. 确认取消操作
4. 任务状态变为 cancelled
```

**场景 3: 重试失败任务**
```
1. 切换到"失败任务"Tab
2. 找到需要重试的任务
3. 点击重试按钮
4. 确认后任务重置为 todo
5. 系统自动重新执行
```

**场景 4: 查找特定任务**
```
1. 展开高级筛选面板
2. 输入关键词搜索
3. 选择任务类型
4. 设置日期范围
5. 表格自动刷新显示结果
```
