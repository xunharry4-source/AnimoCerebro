# 浏览器执行域插件功能说明

- 功能键：`execution:browser`
- 显示名称：浏览器执行域
- 所属插件家族：`execution`
- 实现目录：`src/plugins/execution/`
- 并发模式：单插件
- 当前用途：对浏览器环境进行受控执行
- 默认/回退方向：回退到默认浏览器执行器；仍需强制走 SafetyGate 与 CloudAudit
- 管理红线：不可旁路 CloudAudit
