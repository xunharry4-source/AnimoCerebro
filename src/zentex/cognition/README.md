# Advanced Cognition Engines

This directory contains advanced cognitive engines that go beyond basic reasoning.
- **`social_mind.py`**: Interaction Mind Engine for modeling the intent and knowledge boundaries of interaction partners.
- **`simulation.py`**: High-level simulation logic for counterfactual reasoning.

---

# 高级认知引擎目录

该目录包含超越基础推理的高级认知引擎。
- **`social_mind.py`**: 社会心智引擎（Interaction Mind），用于建模交互对象的意图、知识边界与误解风险。
- **`simulation.py`**: 用于反事实推演的高级模拟逻辑。
