# Zentex Web Console

This directory contains the backend for the Zentex real-time visualization platform.
- **`app.py` / `api.py`**: FastAPI-based web services.
- **`dev_server.py`**: Local development server for visualization.
- **`replay_builder.py`**: Replay mechanism for transcript analysis.
- **`routers/`**: API endpoint definitions for Nine Questions, Transcript flow, and Plugins.

---

# Zentex Web 控制台目录

该目录包含 Zentex 实时可视化平台的后端实现。
- **`app.py` / `api.py`**: 基于 FastAPI 的 Web 服务。
- **`dev_server.py`**: 用于可视化调试的本地开发服务器。
- **`replay_builder.py`**: 用于 Transcript 录像带回放与分析的机制。
- **`routers/`**: 定义九问框架、Transcript 事件流、插件状态等 API 接口。
