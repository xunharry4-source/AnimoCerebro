# Zentex Core

This directory defines the core contracts and base models of the Zentex system. It includes:
- **`plugin_base.py`**: Base classes and lifecycle management for all plugins.
- **`models.py`**: Core data models for the brain runtime state.
- **`cognitive_tools_spec.py`**: Specifications for side-effect-free cognitive tools.
- **`plugin_runtime.py`**: Contracts for plugin loading and health monitoring.

---

# Zentex 核心契约目录

该目录定义了 Zentex 系统的核心契约与基础模型。包含：
- **`plugin_base.py`**: 所有插件的基类及其生命周期管理。
- **`models.py`**: 大脑运行时状态的核心数据模型。
- **`cognitive_tools_spec.py`**: 无副作用认知工具的具体规范。
- **`plugin_runtime.py`**: 插件加载与健康监测的契约定义。
