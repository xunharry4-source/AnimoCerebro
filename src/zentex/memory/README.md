# Memory Management

This directory focuses on memory consolidation and long-term knowledge retention.
- **`consolidation.py`**: Background "sleep-like" mechanism to extract stable patterns and lessons from transcripts.
- **Memory Optimization**: Cleans up noise and focuses on high-value cognitive snapshots.

---

# 记忆管理目录

该目录专注于记忆巩固与长期知识沉淀。
- **`consolidation.py`**: 类似于睡眠的后台机制，用于从 Transcript 中提取稳定的模式与教训（Lessons）。
- **记忆优化**: 清理冗余噪音，专注于高价值的认知快照。
