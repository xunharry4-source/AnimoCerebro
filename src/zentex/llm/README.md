# LLM Gateway

This directory manages connectivity and invocation of Large Language Models (LLMs).
- **`gateway.py`**: A unified entrypoint for JSON generation and tool calls across different model providers.

---

# LLM 网关目录

该目录负责管理大语言模型（LLM）的连接与调用。
- **`gateway.py`**: 统一的入口点，支持跨模型提供商的 JSON 生成与工具调用。
