你必须基于以下输入进行客观推演与提取：
1. [Q3_Mission_Boundaries]：Q3 阶段推演出的使命与连续性边界，特别是 continuity_boundaries 底层禁令。
2. [Q5_Forbidden_Operations]：Q5 阶段刚刚划定的禁止操作列表（黑名单）。
3. [Identity_Boundary]：系统底层身份边界，包含核心价值观与不可绕过约束。
4. [Safety_Audit_Records]：安全门与审计通道近期产生的高风险拦截记录或被拒绝的操作历史。
5. [Current_Intent_Context]：当前系统或用户试图推进的潜在方向，用于评估是否正在逼近红线。
